# archiveweb
Fetch all the URLs that the Google and Wayback Machine knows about for the target removing image, css and other static files

Requirements:

```
▶ python3 -m pip install google
```

Usage:

```
▶ git clone https://github.com/g0ttfr1d/archiveweb

▶ cd archiveweb

▶ python3 archiveweb.py -t xxx.example.com > exemple.txt
```
