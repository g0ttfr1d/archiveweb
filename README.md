# archiveweb
Fetch all the URLs that the Google and Wayback Machine knows about for the target

Requirements:

```
▶ python3 -m pip install google
```

Usage example:

```
▶ git clone http://github.com/g0ttfr1d/archiveweb
▶ cd archiveweb
▶ archiveweb.py -t xxx.example.com > exemple.txt
```
